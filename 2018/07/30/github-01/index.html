Github报错fatal unable to access No such file or directory

2018-07-30    
github

之前有安装过github旧版，一直正常，后来更新了新版，但是git pull的时候如有如下报错：

  • 找到C:\ProgramData\Git\config，打开，修改sslCAinfo（路径得看自己文件相对应的位置）如下：
  • 重点提示：默认的是[http]，如果有必要你需要改成[https]
